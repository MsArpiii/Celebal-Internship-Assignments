# Celebal Technologies Internship Assignments

This repository contains my weekly assignment submissions for the Celebal Technologies Internship Program.

## Week 1 – Data Science Foundations

Topics covered in this assignment:

* NumPy arrays and matrix operations
* Linear Algebra (Vectors, Eigenvalues, Eigenvectors, SVD)
* Data visualization using Matplotlib
* Statistical analysis and hypothesis testing
* Probability distributions (Normal, Binomial, Poisson)
* Bayes' Theorem and Naive Bayes
* Central Limit Theorem (CLT)
* Data drift and Population Stability Index (PSI)
